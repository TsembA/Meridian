About this project
What it is
A production-grade URL shortener built to explore real-world infrastructure — Kubernetes, IaC, observability, and zero-trust security.
How it works
FastAPI backend on k3s, backed by PostgreSQL for storage and Redis for sub-millisecond redirect caching.
Security
Traffic flows exclusively through Cloudflare's WAF. Origin IP is hidden; TLS is strict end-to-end. Secrets live only in AWS SSM.
Infrastructure
Single EC2 node (t3.medium) provisioned by Terraform, deployed via GitHub Actions with OIDC — no static credentials anywhere.
