What is Meridian

A URL shortener that just works

Meridian turns any long, unwieldy link into a clean short URL you can share anywhere — in a message, a presentation, a tweet, or a QR code. Paste your destination, click shorten, and you're done. No sign-up required, no dashboard to navigate, no data sold to anyone.

You can assign a custom code to make links memorable — for example meridiancore.dev/launch instead of a random hash. Every redirect is cached in Redis so links resolve in under a millisecond. Click counts are tracked automatically in the background without slowing anything down.

Under the Hood

Production infrastructure, not a hobby project

Meridian runs on a real production stack. A single EC2 node orchestrated by Kubernetes, deployed through a fully automated CI/CD pipeline with zero static credentials anywhere in the codebase. Every component — networking, secrets, DNS, IAM — is defined as code and reproducible from scratch.

Backend
FastAPI
Async Python API with Pydantic validation and automatic OpenAPI docs
Data layer
PostgreSQL + Redis
Postgres for persistent URL storage, Redis for sub-millisecond redirect caching
Runtime
Kubernetes (k3s)
Lightweight single-node cluster on AWS EC2 t3.medium with self-healing deployments
CDN & Security
Cloudflare
WAF, DDoS protection, strict TLS — origin IP is never exposed to the public internet
CI/CD
GitHub Actions
OIDC auth to AWS, Trivy container scanning, and Helm upgrades on every push to main
Infrastructure as Code
Terraform
VPC, IAM roles, EC2, SSM parameters, and Cloudflare DNS — all reproducible
Observability
Prometheus + Grafana
In-cluster metrics collection and dashboards with Alertmanager for firing alerts
Secrets management
AWS SSM
No secrets in code, images, or environment files — injected at deploy time only
